# compiler
_All **make** commands are to be launched in the src directory_

## compile

### Compile with Cmake:

```bash
cmake . && make
```

### Compile with makefile: 
In the src directory :

```bash
make
```

## Usage


In the src directory :

```bash
./minigcc <file.c> [output]
```

### clean work directory

```bash
make clean
```
